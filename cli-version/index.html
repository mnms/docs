1. LTCLI¶

You can check the version of LTCLI tool.

$ ltcli --version
ltcli version 1.1.5

2. Lightning DB¶

You can check the version of Lightning DB that is deployed in each cluster.

$ ltcli
Cluster '21' selected.
matthew@lightningdb:21> cluster version

- branch: release.flashbase_v1.2.3
